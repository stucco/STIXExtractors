WebGate eDVR Manager ActiveX Controls CVE-2015-2098 Multiple Buffer Overflow Vulnerabilities

WebGate eDVR Manager is prone to multiple buffer-overflow vulnerabilities because it fails to perform boundary checks before copying user-supplied data to insufficiently sized memory buffer.

The controls are identified by CLSID's:

359742AF-BF34-4379-A084-B7BF0E5F34B0

4E14C449-A61A-4BF7-8082-65A91298A6D8

5A216ADB-3009-4211-AB77-F1857A99482C

An attacker can exploit these issues to execute arbitrary code in the context of the application, usually Internet Explorer, using the ActiveX control.Failed attacks will likely cause denial-of-service conditions.
